# 企盟小程序 — 核心业务流程

> 端到端用户旅程

## 流程1：名片分享 → 注册

```
A 分享名片到微信群
  → B 点击名片卡片进入小程序
  → 预加载 A 的公开信息（姓名+头像+一句话身份）
  → 展示 A 信息 + "加好友"按钮
  → 点击加好友 → 弹出注册引导（注明推荐人：A）
  → 手机号填写 → 微信授权 → 获取昵称/头像
  → 注册成功，状态为"普通用户"
  → 自动发送好友申请给 A
  → B 进入通讯录，看到 A 的脱敏信息
  → A 同意 → 双方可见完整信息
```

**涉及模块：** 名片(1) → 通讯录(3) → 用户系统
**涉及API：** POST /api/auth/register → POST /api/friends/request → POST /api/friends/accept

## 流程2：发起活动 → 预约空间

```
进入"活动"Tab → 点击"发起活动"
  → 选择活动模板
  → 填写活动信息（名称、时间、人数、详情）
  → 可选：使用 AI 工具生成海报/文案/流程单
  → 可选：需要预约空间 → 打开空间列表
  → 选择空间 → 查看可预约时段 → 确认预约
  → 设置报名截止时间
  → 发布活动
```

**涉及模块：** 活动(4) → 空间(5) → 支付(9)
**涉及API：** POST /api/activities → POST /api/spaces/{id}/book → POST /api/activities/ai-assist

## 流程3：活动参与 → 签到 → 共同活动圈

```
用户浏览活动 → 报名活动
  → 活动开始 → 现场扫码/点击"签到打卡"
  → 系统标记参与者 → 自动进入该活动共同活动圈
  → 共同活动圈展示所有签到参与者
  → 可一键添加好友（标注"共同参加XXX活动"）
  → 触发互动统计更新（参与者的共同出现次数+1）
```

**涉及模块：** 活动(4) → 通讯录(3) → 互动统计(7)
**涉及API：** POST /api/activities/{id}/register → POST /api/activities/{id}/checkin → GET /api/activities/{id}/circle

## 流程4：付费升级

```
进入"我的" → 点击"升级付费会员"
  → 提交付费申请
  → 系统通知推荐人审核
  → 推荐人在消息中心审核通过/驳回
  → 系统通知站点管理员审核
  → 管理员审核通过/驳回
  → 审核通过 → 进入支付页面 → 微信支付完成
  → 激活全部付费功能
  → 审核驳回 → 退回并说明原因 → 用户可修改后重提
```

**涉及模块：** 付费审核(6) → 支付(9)
**涉及API：** POST /api/membership/apply → POST /api/audit/review → POST /api/payments/membership
