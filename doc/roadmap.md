# 企盟小程序 — 阶段规划与里程碑

> 当前状态：MVP阶段（第1-3周）

## 第一阶段：MVP（第1-3周）

- [ ] 用户系统：微信登录 + 注册（含推荐人绑定）
- [ ] 数字名片：手动录入 + 语音输入 + 基础隐私层级
- [ ] 通讯录：加好友、好友列表
- [ ] 个人中心：名片编辑、我的活动、设置

**交付物：** 可运行的小程序，支持邀请注册和名片展示
**里程碑：** 拉100名种子用户测试

## 第二阶段：核心功能（第4-6周）

- [ ] 活动广场：浏览、发起（付费用户）、报名、签到打卡
- [ ] 共同活动圈：活动关联社交圈
- [ ] 动态状态条：需求/能力发布
- [ ] 付费审核流程：申请 → 双审 → 支付 → 激活
- [ ] 互动统计：共同出现次数 + 详情追溯

**交付物：** 完整社交闭环
**里程碑：** 可发起真实活动，验证场景价值

## 第三阶段：空间+生态（第7-10周）

- [ ] 空间管理：上架、审核、预约系统
- [ ] 空间方结算：平台抽佣、定期结算
- [ ] AI工具集成：活动策划/海报/文案辅助生成
- [ ] 搜索功能（付费用户）
- [ ] 性能优化与提审

**交付物：** 完整平台
**里程碑：** 微信审核通过，正式上线
