# 企盟小程序 — 系统设计（核心）

## 用户角色体系

| 角色 | 权限说明 |
|------|----------|
| 普通用户（免费） | 注册即可，可浏览、加好友、参与免费活动 |
| 付费用户（审核+付费） | 双重审核通过+付费，可发起活动、使用AI工具、空间预约、搜索 |
| 推荐人 | 分享名片的用户，对新人有审核权 |
| 站点管理员 | 属地审核员，审核付费申请中的资质合规 |
| 空间管理者 | 合作方，经平台审核后可上架空间 |
| 超级管理员（平台） | 系统管理、空间统一定价、平台全局配置 |

## 功能模块总览

| 模块 | 描述 |
|------|------|
| 数字名片 📇 | 用户核心身份载体，5个信息板块 + 3种录入方式 + 分级隐私 |
| 动态状态条 📝 | 独立于名片的动态信息流，展示需求/能力，时间线排列 |
| 通讯录与好友 👥 | 推荐注册闭环、加好友、防骚扰、场景化社交规则 |
| 活动广场 🎪 | 浏览/发起（付费）/报名/签到打卡/共同活动圈 |
| 空间预约 🏠 | 空间上架/审核/预约/结算，免费体验1次 |
| 付费审核 💰 | 推荐人审核 → 管理员审核 → 微信支付 → 激活全部功能 |
| 互动统计 📊 | 基于活动签到的共同出现统计，好友间可见 |
| 个人中心 👤 | 名片编辑、动态管理、活动历史、交付记录、数字资产 |
| 支付系统 💳 | 微信支付接入，会员升级+空间预约+未来拓展 |

## 信息架构

小程序采用 **底部Tab导航 + 层级页面结构**，共4个主Tab：

| Tab | 内容 |
|-----|------|
| 名片 📇 | 我的名片展示与编辑入口 |
| 活动 🎪 | 活动广场、发起活动、共同活动圈 |
| 通讯录 👥 | 好友列表、朋友动态、搜索 |
| 我的 👤 | 个人中心、设置、升级付费 |
