# 企盟小程序 — 产品愿景与边界

> 文档不是交付物，而是AI协作的上下文控制系统。
> 所有人/AI在开始任何工作前，先读这个。

## 产品定位

企盟是一个以"人"为核心的**实名制私域社交圈子**小程序。
通过数字名片建立身份，通过推荐制保证质量，通过活动和空间创造真实社交场景，实现高质量人群的连接与协同。

## 核心链条

`数字名片 → 社交关系 → 活动场景 → 空间协同`

## 四大价值主张

| 主张 | 说明 |
|------|------|
| 你的名片会说话 | 数字名片自动生成，语音/文档/手动三种录入，随时分享 |
| 靠谱的人才能进来 | 推荐注册（邀请制）+ 付费审核（双审制），保证用户质量 |
| 在真实场景中连接 | 活动 + 空间，线上线下联动，签到打卡形成共同活动圈 |
| 每一次相遇都有记录 | 互动统计，共同出现次数可追溯，人脉关系清晰可见 |

## 明确不做什么（边界）

- ❌ **不是陌生人社交平台** — 所有关系通过推荐/活动建立
- ❌ **不是大众社交平台** — 实名制、审核制，拒绝匿名/马甲文化
- ❌ **不是内容平台** — 不做信息流、不做短视频，聚焦人脉连接
- ❌ **不是电商平台** — 不做商品交易，空间预约是服务的载体而非目的

## 核心成功指标

| 指标 | 目标 |
|------|------|
| 注册用户数 | 种子期1000人，稳定期10000人 |
| 推荐转化率 | 名片分享→注册转化率 > 15% |
| 活动参与率 | 付费用户月活动参与 > 60% |
| 空间预约率 | 活动关联空间比例 > 30% |
| 互动统计覆盖率 | 活动签到率 > 80% |
