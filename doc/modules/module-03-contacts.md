# 模块3：通讯录与好友系统 👥

> 社交关系管理

## 推荐注册闭环

```
A分享名片到微信群
  → B点击名片卡片进入小程序
  → B看到A的公开信息 + "加好友"按钮
  → B点击加好友 → 弹出注册引导（注明推荐人：A）
  → B微信授权登录 → 填写基本信息
  → 注册成功，状态为"普通用户"，A自动记录为推荐人
  → 自动发送好友申请给A
  → B进入通讯录，看到A的脱敏信息
  → A同意好友申请 → 双方可见完整信息
```

## 加好友规则

| 场景 | 规则 |
|------|------|
| 普通场景 | 不设严格上限，防骚扰机制保护 |
| 活动场景 | 同一活动参与者48h内不限数互加，自动标注"共同参加XX活动" |

## 防骚扰机制

1小时内加好友超过20人 **且** 通过率低于30%，系统弹出提醒（不封禁，仅提醒）。

## 通讯录

- 展示所有已存名片（好友 + 存入但未加好友）
- 好友：显示完整信息，可点击进入详情
- 未好友：显示脱敏信息
- 搜索功能仅付费用户可用
- 朋友动态：展示好友发布的动态状态条

## 相关API

| 方法 | 路径 | 说明 |
|------|------|------|
| GET | /api/friends | 好友列表 |
| POST | /api/friends/request | 发送好友申请 |
| POST | /api/friends/accept | 同意好友申请 |
| GET | /api/friends/suggest | 活动场景好友推荐 |
