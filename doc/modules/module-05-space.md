# 模块5：空间预约 🏠

> 线下场景的核心载体

## 空间管理流程

```
合作方申请 → 平台审核 → 空间管理者上架空间 → 平台统一定价（后续下放定价权限）
```

## 空间信息

名称、地址、容纳人数、设施描述、照片（轮播图）。

## 预约流程

```
活动发起 → 选择"需要预约空间" → 打开空间列表 → 选择场地 → 查看可预约时段 → 确认预约 → 系统锁定 → 活动信息自动关联
```

## 预约状态机

```
待支付 → 已支付（锁定） → 使用中（签到） → 已完成 → 已取消
```

## 用户体验规则

- 普通用户可**免费体验1次**空间预约
- 体验后需付费使用
- 空间预约本身需要付费（区别于活动免费报名）

## 结算

- 平台抽佣（比例后续确定）
- 空间收入定期结算给空间管理者

## 相关API

| 方法 | 路径 | 说明 |
|------|------|------|
| GET | /api/spaces | 空间列表 |
| POST | /api/spaces | 上架空间（空间管理者） |
| POST | /api/spaces/{id}/book | 预约空间 |
| GET | /api/spaces/{id}/slots | 可预约时段 |
| POST | /api/spaces/settle | 空间方结算查询 |
