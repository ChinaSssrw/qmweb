# 模块4：活动广场 🎪

> 活动的浏览、发起、参与、打卡、共同活动圈

## 活动浏览

所有用户可浏览活动列表（卡片列表页）。

## 活动发起（仅付费用户）

- **活动模板**：可选择预设模板（主题、内容结构、报名表单等）
- **关联空间**：如需场地，可选平台已审核的空间
- **活动信息**：主题、内容、时间地点、参与人数上限、报名截止时间
- **AI工具**（付费功能）：
  - AI生成活动流程策划
  - AI生成宣传文案/海报草稿
  - AI生成报名登记表
  - AI生成活动现场流程单

## 活动参与

- 用户可报名参加活动
- 活动开始时扫码或点击"签到打卡"
- 签到后自动进入该活动的共同活动圈

## 共同活动圈

- 同一活动的参与者自动组成的临时社交圈
- 展示所有参与者的脱敏信息
- 可一键添加好友（标注"共同参加XXX活动"）
- 活动结束后48小时内有效

## 打卡机制

活动签到扫码/点击验证 → 标记参与者 → 自动进入共同活动圈 → 触发互动统计更新

## 相关API

| 方法 | 路径 | 说明 |
|------|------|------|
| GET | /api/activities | 活动列表（分页） |
| POST | /api/activities | 发起活动（付费用户） |
| POST | /api/activities/{id}/register | 报名活动 |
| POST | /api/activities/{id}/checkin | 签到打卡 |
| GET | /api/activities/{id}/circle | 共同活动圈 |
| POST | /api/activities/ai-assist | AI辅助生成活动内容 |
