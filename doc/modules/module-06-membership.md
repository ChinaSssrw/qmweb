# 模块6：付费审核与会员体系 💰

> 双重审核 + 微信支付

## 付费审核流程

```
用户选择"升级为付费会员"
  → 提交付费申请
  → 触发双重审核：
      ① 推荐人审核（确认用户身份真实）
      ② 站点管理员审核（确认资质合规）
  → 审核通过 → 微信支付 → 激活全部功能
  → 审核不通过 → 退回申请并说明原因 → 用户可修改后重提
```
