# 模块2：动态状态条 📝

> 独立于名片之外的动态信息流

## 功能描述

用户可发布两种动态内容：
- **我的需求**：当前需要什么（如"寻找合伙人"）
- **我能提供**：当前能帮助什么（如"可提供XX渠道"）

## 技术要点

- 按时间倒序排列
- 支持置顶一条动态
- 好友可见完整动态详情
- 非好友仅见脱敏提示（"该用户有一条动态"）
- 支持点赞、评论、发起私聊
- 支持图文混排

## 相关API

| 方法 | 路径 | 说明 |
|------|------|------|
| GET | /api/status | 获取动态列表 |
| PUT | /api/status | 发布/编辑/置顶动态 |
| DELETE | /api/status/{id} | 删除动态 |
